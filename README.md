# blog

website for [blog](http://linxmusic.github.io/blog) built using [DocPad](http://docpad.org).

## how to

### install

```
git clone https://github.com/linxmusic/blog
npm install
```

### run

`npm start`

### develop

`npm run develop`

to scaffold new post, run `./bin/new`.

posts are stored in "src/documents/posts".

### deploy

`npm run deploy`

# blog

website for [blog](http://linxmix.github.io/blog) built using [DocPad](http://docpad.org).

## how to

### install

```
git clone https://github.com/linxmix/blog
npm install
```

### run

`npm start`

### develop

`npm run develop`

write new post in `src/documents/posts`.

### deploy

`npm run deploy`
